A simple Flutter application that demonstrates how to consume a REST API using the http package. The app fetches and displays posts from the JSONPlaceholder API, showcasing clean separation of concerns using models and repositories, along with proper async data handling using FutureBuilder.

 Features

Fetches posts from a public REST API

Displays data in a scrollable list

Handles loading and error states gracefully

Uses model classes for JSON serialization/deserialization

Repository pattern for API logic

Clean and beginner-friendly Flutter architecture

 Tech Stack

Flutter (Dart)

HTTP package

JSONPlaceholder API
